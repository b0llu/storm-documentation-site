Introduction to Storm

Storm is an open source styling framework, which helps Developers to use best styles and components, by using pre-defined classnames. Storm helps you to quickly build your Dream Project, by focusing much on Functionality.

Installation Guide

To use Storm in your project, use the following link tag, just copy it and paste it in 'head' tag of your html, above all other style sheet

Personal Customization

Storm aims at the satisfaction of user as much as possible but there are times when the pre-defined styles are not sufficient.
In order to override Storm's base classes, user can add there own class to the pre-existing classes, and style them seperately.
