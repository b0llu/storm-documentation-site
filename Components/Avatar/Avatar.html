    • Avatar
    This component consists of user profile pictures, usually found in circular shapes.
    Profile Avatar
    Profile Avatars can be created using the class 'avatar', inside the img tag.
    Sized Avatar
    Size of the avatar can be changed by adding la (11rem) | md (9rem) | sm (5rem) class to existing avatar class
    Square Avatar
    Addind a 'square' class to existing class, will generate a square shaped avatar.
